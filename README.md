# lru-cache
Testing LRU cache  
A simple experiment using [lru-cache](https://www.npmjs.com/package/lru-cache). 

## Dependencies
MongoDB

## Instructions
- ```npm install```
- ```npm run seed```
- ```npm start```  

Take a look at your console. The file of interest is called ```test-cache.js``` if you wanted to play around with it.
